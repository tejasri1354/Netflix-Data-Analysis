# Netflix Data Analysis 📊

## Overview
This project analyzes Netflix movies and TV shows dataset using Python to find trends and insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Movies vs TV Shows analysis
- Content added over years
- Most common genres
- Ratings analysis
- Country-wise content analysis

## Dataset
Netflix dataset in CSV format
